# Fabricore

Fabricore is a high-performance, modern web application built with **TanStack Start** and **Tailwind CSS**. It is designed for speed, scalability, and a premium user experience.

## 🚀 Features

- **Full-Stack React**: Powered by [TanStack Start](https://tanstack.com/start) for seamless SSR, streaming, and server functions.
- **Type-Safe Routing**: Integrated with [TanStack Router](https://tanstack.com/router) for the best-in-class developer experience.
- **Modern Styling**: Built with [Tailwind CSS v4](https://tailwindcss.com/) for lightning-fast, utility-first styling.
- **Premium Components**: UI powered by [Radix UI](https://www.radix-ui.com/) and [Lucide React](https://lucide.dev/).
- **Data Fetching**: Robust state management and caching with [TanStack Query](https://tanstack.com/query).
- **Vercel Ready**: Optimized for deployment on Vercel with Nitro V3 integration.

## 🛠️ Tech Stack

- **Framework**: TanStack Start (React)
- **Engine**: Nitro V3
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Deployment**: Vercel

## 📦 Getting Started

### Prerequisites

- Node.js (v20 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sgpushkar/Fabricore.git
   cd Fabricore
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create a production-ready build:

```bash
npm run build
```

The output will be generated in the `.output` directory, ready for deployment.

## 🚢 Deployment

This project is pre-configured for **Vercel**. To deploy:

1. Push your code to GitHub.
2. Connect your repository to Vercel.
3. Vercel will automatically detect the **TanStack Start** framework and deploy your application.
